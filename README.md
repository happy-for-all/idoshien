# 🚌 移動支援ナビ

移動支援の指定事業所を、現在地・利用者の住所から近い順に検索できるツールサイトです。
対象は「移動支援」のみ（同行援護・行動援護は対象外）。現時点では大阪府のみ対応しています（大阪市＋大阪市以外の市町村、2つのCSVを1つの「大阪府」データとして統合しています）。

## 📁 ファイル構成

```
idoshien/
├ .github/workflows/deploy.yml   ← 手動トリガーのみ（push等では自動実行されない）
├ .github/workflows/cleanup.yml  ← 月1回自動実行（archive/フォルダの古いファイル削除。現状は空振り）
├ index.html                     ← UI本体（検索・フィルター・候補リスト・印刷等）
├ build.py                       ← CSV→ジオコーディング→都道府県別JSON生成
├ csv/
│  ├ 2026_6_1-31.csv             ← 大阪府（大阪市）。ダウンロード時のファイル名のまま
│  └ 2026_6_1-32.csv             ← 大阪府（大阪市以外の市町村）。同上
├ geocode_cache.json             ← 住所→緯度経度のキャッシュ（build.pyが自動生成・更新）
├ cleanup_old_archives.py        ← archive/内の古いファイル削除スクリプト（安全弁、現状空振り）
├ wrangler.json                  ← Cloudflare Worker設定
├ .gitignore
└ README.md                      ← このファイル
```

`dist/`フォルダは`build.py`実行時に自動生成されます（Gitでは管理しません）。

## 🔧 データの更新について

新しい都道府県のCSVを追加する場合：

1. 各都道府県が公開している「移動支援 指定事業所一覧」のCSVを、ダウンロード時のファイル名のまま`csv/`フォルダに追加する
2. `build.py`内の`PREFECTURE_FILES`辞書に、`"ファイル名.csv": ("都道府県名", "スラッグ", 大阪府市町村フィルタが必要か)`の形で1行追加する
   - ファイル名自体には都道府県情報が含まれていないため、**必ず中身を確認してから**正しい都道府県名を追加すること
   - 同じ都道府県内で複数のCSV（例：大阪市／大阪市以外）に分かれている場合は、**同じスラッグ**を指定すれば自動的に1つのデータに統合される
3. 大阪府の「大阪市以外」CSVには、検証の結果、兵庫県・奈良県・京都府など他府県の住所が約15%混入していることが判明したため、大阪府の正式な市町村名（`OSAKA_MUNICIPALITY_PREFIXES`）と照合し、該当しない行を自動的に除外する仕組みを入れている。他の都道府県で同様の混入がある場合は、同じ考え方でフィルタを追加すること

## 🌍 ジオコーディングについて

このCSVには緯度・経度が含まれていないため、`build.py`が国土地理院のジオコーディングAPIを使って住所から座標を求めています。

- 一度変換した住所は`geocode_cache.json`に保存され、次回以降は再利用されます（同じ住所に何度も問い合わせません）
- このAPIは東京大学の研究サービスを基にした準公式・実験的なサービスという位置づけのため、将来的に使えなくなる可能性もゼロではありません。その場合の代替案（デジタル庁のabr-geocoderを関西圏に地域限定して導入する案、郵便番号ベースの簡易座標案）は設計書にまとめてあります

## 🚀 デプロイの手順

このプロジェクトは**完全手動デプロイ**です（GitHub ActionsのRun workflowボタンを押したときだけ実行されます）。

### 初回のみ

```bash
npm install -g wrangler
wrangler login
```

### 毎回のデプロイ手順

1. `csv/`フォルダに最新のCSVが揃っていることを確認
2. GitHubの「Actions」タブ→「Deploy idoshien」→「Run workflow」を押す
3. ワークフローが自動で`build.py`実行→ジオコーディングキャッシュの保存→Cloudflareへのデプロイまで行う
4. 初回デプロイ後、Cloudflareダッシュボードでカスタムドメインを設定する

```
Workers & Pages → idoshien（対象Worker）→「ドメイン」タブ →「+ ドメインを追加」
→ idoshien.pray-power-is-god-and-cocoro.com を入力
```

## ⚠️ 公開前に必ず確認すること

- `index.html`内の`<meta name="robots" content="noindex">`を削除する
- `<link rel="canonical" ...>`のURLが実際の本番URLと一致しているか確認する
- `favicon.ico`をルート直下に設置する（現時点ではリンクタグのみ設置、画像は未設置）

## 📊 データの位置づけ・免責について

掲載データは各都道府県が公表する「移動支援」指定事業所の一覧です。空き状況や対応内容は日々変わるため、必ず電話や公式サイトでご確認ください（`index.html`内にも同様の免責を掲載しています）。
